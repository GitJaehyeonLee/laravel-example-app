## VS Code devcontainer

**Composer 의존성패키지 설치**
```sh
composer install
```
**환경설정 파일**
```sh
cp -rf .env.example.sail .env
```
**VS Code Remote container build**

VS Code - 명령팔레트(`F1`) - `Remote-Containers: Open Folder in Container`
